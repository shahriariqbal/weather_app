# Weather Forecast App


##### used React.js, Chart.js, openweathermap API


# Setup Procedure 

1) Clone this repo <br/>
2) Open command prompt in project folder <br/>
3) Run npm install <br/>
4) Run npm start <br/>

### Live preview link : http://shahriar79.epizy.com/
